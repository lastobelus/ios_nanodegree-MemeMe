MemeEditorViewController

class MemeEditorViewController: UIViewController, UIImagePickerControllerDelegate, UINavigationControllerDelegate, UITextFieldDelegate

A ViewController that provides an interface for creating and editing a Meme. - animates its contents to accomodate the device keyboard when editing texts - provides an image picker - provides an interface for cropping a chosen image - provides an activity controller for sharing a Meme - handles saving an existing meme via unwind segue
