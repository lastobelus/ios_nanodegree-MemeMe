SentMemesTableViewController

class SentMemesTableViewController: UITableViewController, MemesViewer

Displays saved Memes in a table view. - Elements of the grid can be re-ordered by long-pressing an item and moving it around - selecting an item displays it in the Meme Detail view - items can be deleted by swiping left - has an unwind segue action to provide deletion of selected item from the Meme Detail View
